1. RBAC
2. Rate Limiting
3. Payment Gateway menggunakann Midtrans
4. System discount
5. Repository Pattern
6. Service Pattern

Berdasarkan feature" diatas, saya berharap dapat pembelajaran dari feature" yang saya buat dan juga untuk melatih problem solving saya.

=== Untuk Install ===
-Git Clone
-Composer Install
-Npm install
-Composer update
-Php artisan key:generate
-Setting your Env
-Php artisan migrate
-Php artisan db:seed
-Php artisan serve
